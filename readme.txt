Accounting plugin for small business.

== Description ==
WP Ever Accounting is an accounting plugin for easily keep track of income & expense of small companies. Using the plugin can be created multiple types
of expense or income categories so its easier to keep track of the right transaction log. The plugin also have the feature of creating multiple accounts
and ability to transfer funds between these.

== Features ==
* Log income & expense
* Log accounts
* Fund transfer between accounts
* Vendor manage
* Customer manage

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress

= What is the minimum PHP version that I have to maintain to use this plugin? =
To enjoy the sweet features of this powerful plugin you have to at least have PHP version 5.6 or higher installed. If you have a lower version of PHP then you may face problems.
